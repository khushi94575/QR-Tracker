# 📱 Smart QR Tracker

A modern, responsive, and feature-rich **QR Code Generator** built using **HTML, CSS, and JavaScript**. The application allows users to generate QR codes for different types of data, preview them instantly, and download them for future use.

## 📌 Project Overview

Smart QR Tracker is a web application designed to simplify QR code generation for various types of information. It features a modern glassmorphism interface, dynamic forms, real-time QR generation, and a clean user experience.

This project demonstrates frontend development concepts including DOM manipulation, event handling, dynamic UI rendering, form validation, QR code generation, and responsive design.
